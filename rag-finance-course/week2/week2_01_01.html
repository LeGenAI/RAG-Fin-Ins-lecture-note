📊 메타데이터(Metadata)란?

정의

메타데이터는 "데이터에 대한 데이터"로, 원본 데이터의 특성, 구조, 의미를 설명하는 정보입니다.

🏷️ 쉬운 비유: 도서관의 도서카드

책(원본 데이터) 자체가 아니라, 책의 제목, 저자, 출간연도, 분류번호 등이 적힌 카드가 메타데이터입니다. 이 정보로 원하는 책을 빠르게 찾을 수 있죠!

📝 Markdown(.md) 형식 변환

Markdown이란?

간단한 문법으로 구조화된 문서를 만들 수 있는 경량 마크업 언어입니다. AI가 이해하기 쉽고, 사람이 읽기도 편합니다.

💳 신용카드 사용 내역 → Markdown 변환 예시
# 2024년 1월 신용카드 사용 내역 ## 식비 관련 - **2024-01-15**: 스타벅스 - 6,500원 (카페) - **2024-01-16**: 맥도날드 - 8,900원 (패스트푸드) ## 교통비 관련 - **2024-01-17**: 지하철 - 2,000원 (대중교통) - **2024-01-18**: 택시 - 12,000원 (야간 긴급) ## 쇼핑 관련 - **2024-01-19**: 이마트 - 85,000원 (생필품)

✅ Markdown의 장점

  • 구조화된 정보로 AI가 카테고리별로 이해 가능
  • 사람이 읽기 쉬워 검토 및 수정이 용이
  • 다른 형식으로 쉽게 변환 가능

🗃️ JSON 형식 변환

JSON이란?

JavaScript Object Notation의 약자로, 데이터를 키-값 쌍으로 저장하는 형식입니다. AI가 정확하게 데이터를 처리할 수 있습니다.

🏦 보험 계약 정보 → JSON 변환 예시
{ "보험정보": { "계약번호": "ABC-2024-001", "상품명": "종합건강보험", "보험사": "삼성화재", "가입일": "2024-01-01", "보험료": { "월보험료": 85000, "납입방법": "자동이체", "납입일": 15 }, "보장내용": { "입원비": "일당 10만원", "수술비": "최대 500만원", "암진단": "2000만원" }, "수익자": "홍길동", "태그": ["건강", "가족보장", "필수보험"] } }

✅ JSON의 장점

  • 정확한 데이터 구조로 AI가 오해 없이 처리 가능
  • 복잡한 데이터 관계도 표현 가능
  • 프로그래밍 언어에서 바로 사용 가능
